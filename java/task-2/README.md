# JAVA基本语法与程序流程控制

> 需在`/tasks/java`目录下创建自己的文件夹,并建立自己的目录再建立`task-2`目录并在其中提交任务

### 任务

编程，程序读取一个输入，输入为棱形的行数（行数>=3且为奇数），然后输出一个棱形，如下形式：  
```
   O
  OXO
 OXOXO
OXOXOXO
 OXOXO
  OXO
   O
```
其中，每一行的前面用空格填充，每一行的第奇数个字符为O，第偶数个字符为X。
