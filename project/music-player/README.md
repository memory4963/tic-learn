# Android端的音乐播放器

> 难度：××××××

### 要求

* 音乐列表
* 歌手列表
* 专辑列表
* 播放音乐，进度控制，歌词显示
* 专辑图片
* 界面友好、流畅
* 收藏歌曲
* 通知栏显示正在播放的歌曲
* 后台播放
* 电话响铃等情况暂停播放
* 播放模式控制（顺序播放、随机等）
* 可自行增加功能

### 非功能性要求

* 流畅
* 使用Fragment
* 遵循Material Design
* 模块化设计

### 知识点

* MediaPlayer的使用
* Service
* ListVie/RecyclerView
* Fragment
* MediaStore
* Sqlite数据库查询
* 。。。太多了
